# SuperMED Frontend

Frontend systemu SuperMED — Project MED-CORE.

Aplikacja jest przygotowana jako panel webowy dla Systemu Zarządzania Szpitalem i Elektroniczną Dokumentacją Medyczną SuperMED.

## Technologie

- Angular
- TypeScript
- Reactive Forms
- Angular Router
- JWT Interceptor
- Mock services do czasu podłączenia backendu

## Uruchomienie

```powershell
npm.cmd install
npm.cmd start
